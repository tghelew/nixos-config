* Installation
1. Install nix on target machine
   #+begin_src sh
 sh <(curl -L https://nixos.org/nix/install)
   #+end_src
2. Clone this repository to ~$HOME/.config/~
3. Run the following command
   #+begin_src sh
cd ~/.config/nixos-config
./bin/tnix rebuilt -t darwin -host <hostname>
   #+end_src
